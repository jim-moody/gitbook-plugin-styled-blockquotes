## Markdown Within Blockquote

You can still use standard markdown inside a blockquote and it will render as expected

**Header**

```markdown
> **info**
> ### Header
> Content here
```

> **info**
> ### Header
> Content here

**List**

```markdown
> **info**
> * Item 1
> * Item 2
```

> **info**
> * Item 1
> * Item 2
